Условия использования GoodbyeYT

Последнее обновление: 2024 год

1. Общие положения

GoodbyeYT — независимый некоммерческий проект, предоставляющий альтернативный интерфейс для доступа к видеоконтенту платформы YouTube через открытые публичные инстансы (Piped, Cobalt). Мы не являемся аффилированными с Google LLC или YouTube.

2. Использование YouTube API

Приложение использует ваш личный API-ключ YouTube Data API v3, который вы предоставляете добровольно. Ключ хранится только в вашем браузере (localStorage) и не передаётся на наши серверы.

Использование YouTube API подчиняется Условиям использования YouTube API и Политике конфиденциальности Google.

3. Правила использования

  • Использование GoodbyeYT разрешено только в личных некоммерческих целях
  • Запрещено использование для незаконных целей
  • Запрещено нарушение авторских прав на контент
  • Мы не несём ответственности за контент, размещённый на YouTube
  • Пользователь самостоятельно несёт ответственность за соблюдение законодательства своей страны

4. Открытые инстансы

Видеостримы доставляются через открытые публичные инстансы проектов Piped и Cobalt. Качество и доступность стримов зависит от этих третьих сторон. Мы не несём ответственности за их работоспособность.

5. Ограничение ответственности

GoodbyeYT предоставляется «как есть», без каких-либо гарантий. Мы не несём ответственности за любые убытки, возникшие в результате использования или невозможности использования сервиса.

6. Интеллектуальная собственность

GoodbyeYT является открытым проектом. Авторские права на видеоконтент принадлежат соответствующим правообладателям. Мы не претендуем на права на контент YouTube.

7. Изменения условий

Мы оставляем за собой право изменять эти условия. Продолжение использования сервиса означает согласие с новыми условиями.

8. Применимое право

Настоящие условия регулируются законодательством Российской Федерации.

Политика конфиденциальности GoodbyeYT

Последнее обновление: 2024 год

1. Какие данные мы собираем

Мы не собираем персональные данные. GoodbyeYT работает как прокси-клиент.

  • API-ключ YouTube: хранится только в вашем браузере (localStorage), не передаётся на наши серверы
  • Настройки: тема, язык, качество видео — хранятся локально в браузере
  • Логи сервера: анонимные технические логи (IP, время запроса) для диагностики, без привязки к личности

2. Как используются данные

  • API-ключ используется исключительно для запросов к YouTube Data API v3 от вашего имени
  • Мы не продаём, не передаём, не обрабатываем ваши данные в коммерческих целях
  • Мы не используем аналитику, трекеры и рекламные сети

3. Cookie

Мы используем только технически необходимые cookie (сессия). Рекламные и трекинговые cookie отсутствуют.

4. Третьи стороны

При воспроизведении видео запросы могут направляться к публичным инстансам Piped/Cobalt и серверам YouTube/Google. На их политику конфиденциальности мы не влияем.

5. Безопасность данных

Мы применяем стандартные меры безопасности (HTTPS, заголовки безопасности). Однако абсолютная безопасность в интернете невозможна.

6. Ваши права

Вы можете в любой момент очистить все локальные данные через «Настройки → Сбросить данные».

Кто мы?

GoodbyeYT — независимый некоммерческий проект, созданный энтузиастами. Мы не связаны с Google, YouTube, Роскомнадзором или какими-либо государственными структурами.

Наша миссия

Дать российским пользователям удобный, быстрый и безопасный доступ к YouTube-контенту, используя открытые технологии и публичные API.

Технологии

  • 🔴 YouTube Data API v3 — поиск и метаданные
  • ⚡ Piped — открытый фронтенд YouTube для стриминга
  • 🔗 Cobalt — открытый инструмент для работы с медиа
  • 🛡️ Node.js + Express — бэкенд

Открытый исходный код

Весь код GoodbyeYT открыт и доступен на GitHub. Мы приветствуем вклад сообщества.
